## 📖 简介

本次更新为 AI 图片管理系统添加了完整的**厂商和模型管理功能**。用户现在可以在设置页面中自由管理所有 AI 图片生成厂商和对应的模型，无需修改代码或数据库。

---

## ✨ 核心功能

### 🏢 厂商管理
- ✅ 查看所有厂商
- ✅ 添加新厂商
- ✅ 编辑厂商信息（名称、路径标识）
- ✅ 删除厂商（软删除）
- ✅ 展开/折叠查看模型

### 🤖 模型管理
- ✅ 查看厂商下的所有模型
- ✅ 添加新模型
- ✅ 编辑模型信息（名称、路径标识、描述）
- ✅ 删除模型（软删除）

### 📊 数据展示
- ✅ 显示所有厂商（包括没有图片的）
- ✅ 显示每个厂商的模型数量
- ✅ 显示每个模型的图片数量
- ✅ 实时更新，无需刷新

## 🚀 快速开始

### 1. 打开厂商管理

**方式一**：按键盘快捷键
```
Ctrl + ,
```

**方式二**：点击设置图标
```
点击应用右上角的 ⚙️ 图标
```

### 2. 进入厂商管理标签

在设置窗口中，点击顶部的 **"厂商管理"** 标签页。

### 3. 开始管理

- **添加厂商**：点击"添加新厂商"按钮
- **编辑厂商**：点击厂商右侧的 ✏️ 图标
- **删除厂商**：点击厂商右侧的 🗑️ 图标
- **添加模型**：展开厂商后点击"添加模型"按钮

## 🎬 使用示例

### 示例 1：添加 Stable Diffusion 厂商

```
1. 打开设置 → 厂商管理
2. 点击"添加新厂商"
3. 输入：
   - 厂商名称：Stable Diffusion
   - 路径标识：stable-diffusion
4. 点击 ✓ 保存
```

### 示例 2：为 Stable Diffusion 添加 SDXL 模型

```
1. 展开 Stable Diffusion 厂商
2. 点击"添加模型"
3. 输入：
   - 模型名称：SDXL 1.0
   - 路径标识：sdxl-1.0
4. 点击 ✓ 保存
```

### 示例 3：修改厂商名称

```
1. 找到要修改的厂商
2. 点击右侧的 ✏️ 编辑图标
3. 修改名称
4. 点击 ✓ 保存
```

## 💡 最佳实践

### 路径标识命名规范

✅ **推荐**：
- `openai`
- `google-imagen`
- `stable-diffusion`
- `midjourney-v6`

❌ **不推荐**：
- `OpenAI`（包含大写）
- `google imagen`（包含空格）
- `stable_diffusion`（使用下划线）
- `图片生成器`（使用中文）

### 数据管理建议

1. **先配置后使用**：建议在导入图片前配置好厂商和模型
2. **谨慎删除**：删除是软删除，但仍建议谨慎操作
3. **避免修改路径标识**：路径标识一旦创建，建议不要修改

### 常见问题

**Q: 删除厂商后还能恢复吗？**  
A: 可以。删除是软删除，数据仍保留在数据库中，可以联系技术支持恢复。

**Q: 修改厂商名称会影响已有图片吗？**  
A: 不会。图片记录的是厂商 ID，不是名称。

**Q: 路径标识可以修改吗？**  
A: 可以，但不建议。修改后可能需要重新分类相关图片。
