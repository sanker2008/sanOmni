# 删除功能说明

## 更新内容

### 1. 图片预览修复
- 在 `src-tauri/tauri.conf.json` 中配置了 CSP 和 asset 协议
- 在 `src-tauri/Cargo.toml` 中添加了 `protocol-asset` 特性
- 在 `src-tauri/capabilities/default.json` 中添加了 `fs:allow-read-file` 权限

### 2. 删除功能
已为待整理添加了完整的删除功能：

#### 单个图片删除
- 在 `ImageCard` 组件的悬停菜单中添加了删除按钮（垃圾桶图标）
- 点击删除按钮会弹出确认对话框
- 确认后会调用后端 API 删除图片记录
- 删除成功后会从界面中移除该图片

#### 批量删除
- 在待整理工具栏中添加了"批量删除"按钮
- 选中多张图片后可以批量删除
- 会显示删除确认对话框，提示删除数量
- 删除完成后会显示成功和失败的数量

## 使用方法

### 删除单张图片
1. 将鼠标悬停在图片卡片上
2. 点击右下角的垃圾桶图标
3. 在确认对话框中点击"确定"

### 批量删除图片
1. 选中要删除的图片（点击图片左上角的圆圈）
2. 点击工具栏中的"批量删除"按钮
3. 在确认对话框中点击"确定"

## 注意事项
- 删除操作不可恢复，请谨慎操作
- 删除只会删除数据库记录，不会删除磁盘上的原始文件
- 如果需要同时删除磁盘文件，需要在后端 API 中实现
