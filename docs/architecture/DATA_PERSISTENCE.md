# 数据持久化说明

## 重新安装应用后数据是否会丢失？

### ❌ 不会丢失！

应用数据存储在 **用户数据目录**，而不是应用安装目录。

## 存储位置对比

### 应用安装目录（会被删除）
```
C:\Program Files\sanOmni\          # 应用程序文件
├── sanOmni.exe
├── resources\
└── ...
```
**重新安装时：** 这个目录会被删除并重新创建

### 用户数据目录（不会被删除）
```
C:\Users\<用户名>\AppData\Roaming\com.sanomni.app\
├── data\
│   └── database.sqlite          # 数据库
├── inbox\                        # 待整理图片
│   └── ...
└── archived\                     # 归档图片
    └── ...
```
**重新安装时：** 这个目录**完全不受影响**

## 为什么不会丢失？

### 1. 使用 Tauri 的 `appDataDir()`
```typescript
const { appDataDir } = await import("@tauri-apps/api/path");
const appDir = await appDataDir();
// 返回: C:\Users\<用户名>\AppData\Roaming\com.sanomni.app\
```

这个目录是：
- ✅ 用户级别的数据目录
- ✅ 独立于应用安装位置
- ✅ 卸载应用时不会被删除
- ✅ 重新安装后会继续使用同一个目录

### 2. Windows 标准做法
Windows 应用通常将用户数据存储在：
- `%APPDATA%` (Roaming) - 用于漫游数据
- `%LOCALAPPDATA%` (Local) - 用于本地数据

这些目录在卸载/重装应用时都会保留。

## 各种场景下的数据保留情况

### ✅ 重新安装应用
- **数据库：** 保留
- **inbox 图片：** 保留
- **archived 图片：** 保留
- **设置：** 保留

### ✅ 升级应用版本
- **所有数据：** 完全保留
- **数据库：** 可能需要迁移（如果数据库结构变化）

### ✅ 卸载应用
- **应用程序：** 删除
- **用户数据：** **保留**（需要手动删除）

### ❌ 手动删除用户数据目录
如果用户手动删除了：
```
C:\Users\<用户名>\AppData\Roaming\com.sanomni.app\
```
那么所有数据会丢失。

### ❌ 重装 Windows 系统
- 如果没有备份用户文件夹，数据会丢失
- 建议定期备份或使用云同步

## 如何完全清理应用数据

### 方法1：手动删除
1. 卸载应用
2. 删除用户数据目录：
   ```
   %APPDATA%\com.sanomni.app
   ```

### 方法2：在应用中提供"重置"功能
可以在设置中添加"重置应用数据"按钮，清空：
- 数据库
- inbox 目录
- archived 目录

## 数据备份建议

### 重要数据位置
```
C:\Users\<用户名>\AppData\Roaming\com.sanomni.app\
├── data\database.sqlite          # 最重要：包含所有元数据
├── archived\                     # 重要：已整理的图片
└── inbox\                        # 可选：临时图片
```

### 备份策略
1. **定期备份数据库**
   - 复制 `data\database.sqlite` 到安全位置
   - 或使用 SQLite 的备份命令

2. **备份归档图片**
   - 复制整个 `archived\` 目录
   - 或使用云同步服务

3. **inbox 可以不备份**
   - 这些是临时副本
   - 原始文件还在原位置

## 跨设备同步

### 当前状态
- ❌ 不支持自动同步
- 数据存储在本地

### 未来可能的方案
1. **云存储集成**
   - 将 archived 目录设置为云盘文件夹
   - OneDrive、Google Drive、Dropbox 等

2. **数据库同步**
   - 使用云数据库（如 Supabase）
   - 或定期同步 SQLite 文件

3. **导出/导入功能**
   - 导出数据库和图片为压缩包
   - 在另一台设备上导入

## 常见问题

### Q: 卸载应用后数据还在吗？
**A:** 是的，数据保留在 `%APPDATA%\com.sanomni.app\`

### Q: 重装系统后数据会丢失吗？
**A:** 是的，除非你备份了用户文件夹

### Q: 如何迁移到新电脑？
**A:** 复制整个 `%APPDATA%\com.sanomni.app\` 目录到新电脑的相同位置

### Q: 数据占用多少空间？
**A:** 
- 数据库：通常几 MB
- inbox：取决于未归档的图片数量
- archived：取决于已归档的图片数量

### Q: 可以更改数据存储位置吗？
**A:** 当前不支持，但可以添加此功能：
- 在设置中指定自定义路径
- 使用符号链接重定向到其他磁盘
