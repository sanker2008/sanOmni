# 内嵌浏览器功能

## 概述

sanOmni 现在支持内嵌浏览器功能，允许你在应用内直接访问 AI 图片生成网站，生成图片后可以直接拖拽到应用中导入。

## 功能特点

### 1. 预设网站快速访问

应用内置了常用的 AI 图片生成网站：

- **Gemini** (https://gemini.google.com/app)
  - Google 的 AI 助手，支持图片生成
  
- **DALL-E / ChatGPT** (https://chatgpt.com)
  - OpenAI 的 ChatGPT，支持 DALL-E 图片生成
  
- **Midjourney** (https://www.midjourney.com)
  - 专业的 AI 图片生成平台

### 2. 自定义网址访问

除了预设网站，你还可以：
- 输入任意网址访问
- 自动添加 `https://` 协议（如果未指定）
- 支持所有标准网页

### 3. 浏览器控制

- **后退/前进** - 导航历史（当前版本暂不支持）
- **主页** - 返回欢迎页面
- **刷新** - 重新加载当前页面
- **外部浏览器打开** - 在系统默认浏览器中打开当前页面

## 使用方法

### 访问内嵌浏览器

1. 点击顶部导航栏的 **"浏览器"** 标签
2. 选择预设网站或输入自定义网址
3. 点击 **"访问"** 按钮或按 Enter 键

### 工作流程示例

1. **打开 Gemini**
   - 点击 "Gemini" 预设按钮
   - 在 Gemini 中生成图片
   - 下载图片到本地

2. **导入到应用**
   - 切换到 "待整理" 标签
   - 将下载的图片拖拽到应用中
   - 或使用 "导入" 按钮选择文件

3. **整理和管理**
   - 添加标签、提示词等元数据
   - 归档到相应的厂商/模型文件夹

## 技术说明

### iframe 沙箱

内嵌浏览器使用 HTML iframe 实现，配置了以下沙箱权限：

```html
sandbox="allow-same-origin allow-scripts allow-forms allow-popups allow-popups-to-escape-sandbox"
```

这确保了：
- ✅ 网页可以正常运行 JavaScript
- ✅ 支持表单提交和登录
- ✅ 允许弹出窗口（如 OAuth 登录）
- ⚠️ 有一定的安全隔离

### 限制和注意事项

1. **跨域限制**
   - 某些网站可能禁止在 iframe 中加载（X-Frame-Options）
   - 如果网站无法加载，请使用 "外部浏览器打开" 功能

2. **功能限制**
   - 当前版本不支持浏览器历史记录（后退/前进）
   - 不支持多标签页
   - 不支持书签管理

3. **性能考虑**
   - 内嵌浏览器会占用额外内存
   - 建议在使用完毕后切换到其他标签页

4. **安全建议**
   - 只访问可信的网站
   - 注意保护个人信息和登录凭证
   - 定期清理浏览器缓存

## 常见问题

### Q: 为什么某些网站无法加载？

A: 某些网站（如银行、支付网站）出于安全考虑禁止在 iframe 中加载。请使用 "外部浏览器打开" 功能。

### Q: 可以直接从网页拖拽图片到应用吗？

A: 当前版本暂不支持。请先下载图片，然后通过 "导入" 功能添加到应用。

### Q: 浏览器会保存我的登录状态吗？

A: 会的。iframe 使用浏览器的标准存储机制，登录状态会被保存。

### Q: 如何清除浏览器数据？

A: 当前版本需要清除整个应用的数据。未来版本会添加专门的清除功能。
