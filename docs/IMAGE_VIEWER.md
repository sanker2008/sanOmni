# 图片查看器功能

## 📋 功能说明

添加了一个全功能的图片查看器，可以查看大图并显示详细信息。

## ✨ 主要特性

### 1. 大图查看
- **高清显示** - 以原始尺寸显示图片
- **自适应缩放** - 自动适应窗口大小
- **黑色背景** - 更好的查看体验

### 2. 图片导航
- **左右箭头按钮** - 点击切换上一张/下一张
- **键盘导航** - 使用 ← → 键快速切换
- **进度显示** - 显示当前位置（如：3 / 10）

### 3. 详细信息侧边栏
- **模型信息** - 显示所有关联的模型
- **标签** - 显示所有标签
- **Prompt** - 完整的生成提示词
- **Negative Prompt** - 负面提示词
- **水印信息** - 水印检测结果
- **文件信息** - 格式、尺寸、大小、状态
- **文件路径** - 完整的文件路径

### 4. 快捷操作
- **编辑** - 快速打开编辑弹窗
- **另存为** - 保存图片到其他位置
- **外部打开** - 用系统默认程序打开
- **关闭** - 关闭查看器

## 🎯 使用方法

### 打开查看器
**方式一：双击图片**
1. 在收件箱或已归档视图中
2. 双击任意图片卡片
3. 自动打开查看器

**方式二：单击选择后双击**
1. 单击选择图片
2. 再次双击打开查看器

### 导航操作
- **下一张**：点击右侧箭头按钮 或 按 → 键
- **上一张**：点击左侧箭头按钮 或 按 ← 键
- **关闭**：点击 X 按钮 或 按 ESC 键

### 快捷操作
- **编辑图标** - 打开快速编辑弹窗
- **下载图标** - 另存为到其他位置
- **外部链接图标** - 用系统默认程序打开
- **X 图标** - 关闭查看器

## ⌨️ 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| ← | 上一张图片 |
| → | 下一张图片 |
| ESC | 关闭查看器 |

- **Info Sidebar** - 详细信息侧边栏
- **Footer** - 快捷键提示

### 状态管理

#### UIStore 新增状态
```typescript
interface UIStore {
  isImageViewerOpen: boolean;      // 查看器是否打开
  viewingImageId: string | null;   // 当前查看的图片 ID
  
  openImageViewer: (imageId: string) => void;   // 打开查看器
  closeImageViewer: () => void;                  // 关闭查看器
  setViewingImageId: (imageId: string) => void; // 切换图片
}
```

### 键盘导航实现

```typescript
useEffect(() => {
  if (!isImageViewerOpen) return;

  const handleKeyDown = (e: KeyboardEvent) => {
    if (e.key === "ArrowLeft") {
      goToPrevious();
    } else if (e.key === "ArrowRight") {
      goToNext();
    } else if (e.key === "Escape") {
      closeImageViewer();
    }
  };

  window.addEventListener("keydown", handleKeyDown);
  return () => window.removeEventListener("keydown", handleKeyDown);
}, [isImageViewerOpen, currentIndex]);
```

### 图片导航逻辑

```typescript
// 根据当前标签页获取图片列表
const currentImages = activeTab === "inbox" ? inboxImages : archivedImages;

// 查找当前图片的索引
const currentIndex = currentImages.findIndex((img) => img.id === viewingImageId);

// 导航到上一张
const goToPrevious = () => {
  if (currentIndex > 0) {
    setViewingImageId(currentImages[currentIndex - 1].id);
  }
};

// 导航到下一张
const goToNext = () => {
  if (currentIndex < currentImages.length - 1) {
    setViewingImageId(currentImages[currentIndex + 1].id);
  }
};
```

## 💡 设计考虑

### 为什么使用双击打开？

**原因**：
1. **避免冲突** - 单击用于选择图片
2. **符合习惯** - 双击打开是通用操作
3. **防止误触** - 双击需要明确意图

### 为什么添加侧边栏？

**优点**：
1. **信息完整** - 显示所有相关信息
2. **不遮挡图片** - 侧边栏不影响图片查看
3. **可滚动** - 信息多时可以滚动查看
4. **结构清晰** - 分类显示各种信息

### 为什么支持键盘导航？

**优点**：
1. **效率高** - 键盘操作比鼠标快
2. **符合习惯** - 左右箭头是通用导航键
3. **单手操作** - 可以单手浏览图片
4. **专业感** - 类似专业图片管理软件
