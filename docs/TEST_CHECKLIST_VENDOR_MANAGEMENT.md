# 厂商管理功能测试清单

## 测试环境准备

- [ ] 确保应用已编译并运行
- [ ] 打开应用并进入设置页面（Ctrl + ,）
- [ ] 切换到"厂商管理"标签页

---

## 1. 查看功能测试

### 1.1 查看厂商列表
- [ ] 能看到所有已配置的厂商
- [ ] 每个厂商显示名称和模型数量
- [ ] 厂商按字母顺序排列

### 1.2 展开/折叠厂商
- [ ] 点击厂商左侧的箭头图标可以展开
- [ ] 展开后显示该厂商下的所有模型
- [ ] 再次点击可以折叠

### 1.3 查看模型列表
- [ ] 展开厂商后能看到所有模型
- [ ] 每个模型显示名称

---

## 2. 添加功能测试

### 2.1 添加厂商
- [ ] 点击"添加新厂商"按钮
- [ ] 输入厂商名称：`测试厂商`
- [ ] 输入路径标识：`test-vendor`
- [ ] 点击保存图标
- [ ] 新厂商出现在列表中
- [ ] 显示"0 个模型"

### 2.2 添加模型
- [ ] 展开刚添加的"测试厂商"
- [ ] 点击"添加模型"按钮
- [ ] 输入模型名称：`测试模型 1`
- [ ] 输入路径标识：`test-model-1`
- [ ] 点击保存图标
- [ ] 新模型出现在厂商下方
- [ ] 厂商显示"1 个模型"

### 2.3 添加多个模型
- [ ] 继续添加第二个模型：`测试模型 2`
- [ ] 路径标识：`test-model-2`
- [ ] 厂商显示"2 个模型"

---

## 3. 编辑功能测试

### 3.1 编辑厂商
- [ ] 点击"测试厂商"右侧的编辑图标
- [ ] 修改名称为：`测试厂商（已修改）`
- [ ] 点击保存图标
- [ ] 名称更新成功
- [ ] 点击 X 图标可以取消编辑

### 3.2 编辑模型
- [ ] 展开"测试厂商（已修改）"
- [ ] 点击"测试模型 1"右侧的编辑图标
- [ ] 修改名称为：`测试模型 1（已修改）`
- [ ] 点击保存图标
- [ ] 名称更新成功

---
