## 🚀 启动测试

### 步骤 1：启动开发服务器

```bash
npm run tauri:dev
```

### 步骤 2：导航到 Prompt 管理

1. 启动应用
2. 点击顶部导航栏的"Prompt 管理"标签页（✨ 图标）

### 步骤 3：测试功能

#### 测试 1：创建 Prompt 组

1. 点击"添加 Prompt"按钮
2. 输入 Prompt 信息
3. 测试筛选功能：
   - 点击"全部"查看所有图片
   - 点击"已关联"查看已分组图片
   - 点击"未关联"查看未分组图片
4. 选择图片
5. 保存

**预期结果**：
- ✅ Prompt 组创建成功
- ✅ 图片正确关联
- ✅ 列表中显示新组
- ✅ 显示正确的图片数量

#### 测试 2：筛选功能

1. 打开创建或编辑对话框
2. 点击"全部"按钮
   - 应显示所有图片
3. 点击"已关联"按钮
   - 只显示已关联到其他组的图片
   - 显示"X 个组"徽章
4. 点击"未关联"按钮
   - 只显示未关联的图片
   - 无"X 个组"徽章

**预期结果**：
- ✅ 筛选按钮正确切换
- ✅ 激活按钮高亮显示
- ✅ 图片列表正确筛选
- ✅ 徽章正确显示

#### 测试 3：搜索 + 筛选

1. 选择筛选模式（如"已关联"）
2. 在搜索框输入关键词
3. 观察结果

**预期结果**：
- ✅ 在筛选结果中进行搜索
- ✅ 同时满足筛选和搜索条件

#### 测试 4：编辑 Prompt 组

1. 点击某个 Prompt 组的"编辑"按钮
2. 修改 Prompt 信息
3. 使用筛选功能调整关联图片
4. 保存

**预期结果**：
- ✅ 加载现有数据
- ✅ 显示已关联的图片（已选中状态）
- ✅ 保存后更新成功

#### 测试 5：查看 Prompt 组详情

1. 点击某个 Prompt 组的"查看"按钮
2. 查看图片按模型分组显示

**预期结果**：
