# 厂商管理快捷入口

## 📋 功能说明

在已归档视图的"厂商分类"标题旁边添加了一个编辑图标按钮，点击后可以快速打开设置页面的厂商管理标签。

## ✨ 功能特性

### 快捷入口
- **位置**：已归档视图 → 左侧边栏 → "厂商分类"标题右侧
- **图标**：编辑图标（铅笔）
- **功能**：点击后直接打开设置页面并切换到"厂商管理"标签

### 使用场景
1. **快速添加厂商** - 看到新的厂商时，点击图标快速添加
2. **编辑厂商信息** - 需要修改厂商名称或路径时
3. **管理模型** - 为厂商添加或编辑模型
4. **删除厂商** - 删除不需要的厂商

## 🎯 使用方法

### 方式一：通过快捷入口
1. 切换到"已归档"标签
2. 在左侧边栏找到"厂商分类"标题
3. 点击标题右侧的编辑图标（铅笔）
4. 自动打开设置页面并切换到"厂商管理"标签

### 方式二：通过设置页面
1. 按 `Ctrl + ,` 打开设置
2. 点击"厂商管理"标签

## 🔧 技术实现

### 前端变更

#### ArchivedView.tsx
1. **添加图标导入**
   ```tsx
   import { Edit2 } from "lucide-react";
   ```

2. **添加 UIStore 方法**
   ```tsx
   const { openSettings, setSettingsTab } = useUIStore();
   ```

3. **添加编辑按钮**
   ```tsx
   <Button
     variant="ghost"
     size="icon"
     className="h-6 w-6"
     onClick={() => {
       setSettingsTab("vendors");
       openSettings();
     }}
     title="管理厂商"
   >
     <Edit2 className="w-3.5 h-3.5" />
   </Button>
   ```

### UI 设计

#### 按钮样式
- **大小**：`h-6 w-6` - 小巧不占空间
- **变体**：`ghost` - 透明背景，悬停时显示
- **图标大小**：`w-3.5 h-3.5` - 与标题文字大小协调
- **提示**：`title="管理厂商"` - 悬停显示提示

#### 布局调整
- 标题区域从 `p-3 border-b` 改为 `p-3 border-b flex items-center justify-between`
- 标题和按钮之间自动分布空间
- 保持整体视觉平衡

## 💡 设计考虑

### 为什么放在这里？

**优点**：
1. **上下文相关** - 在查看厂商列表时，自然想到管理厂商
2. **位置显眼** - 标题旁边，容易发现
3. **不占空间** - 小图标不影响布局
4. **符合习惯** - 编辑图标是通用的管理入口

**替代方案对比**：

| 方案 | 优点 | 缺点 | 选择 |
|------|------|------|------|
| 标题旁边的图标 | 显眼、上下文相关 | - | ✅ 采用 |
| 每个厂商旁边的图标 | 可以直接编辑特定厂商 | 占用空间、视觉混乱 | ❌ |
| 底部的管理按钮 | 明确的操作 | 不够显眼、占用空间 | ❌ |
| 右键菜单 | 不占空间 | 不够直观、可发现性差 | ❌ |

### 为什么只在已归档视图？

**原因**：
1. 已归档视图有厂商分类侧边栏
2. 待整理视图没有厂商分类功能
3. 保持功能的一致性和相关性
