# Prompt 对比功能 - 快速开始

## 5 分钟上手指南

### 第一步：准备图片

确保你的图片包含 prompt 信息。导入图片时，系统会尝试从文件名或元数据中提取 prompt。

**示例**：
```
图片 A: prompt = "a beautiful sunset over mountains"
图片 B: prompt = "a beautiful sunset over mountains" (相同)
图片 C: prompt = "a beautiful sunset over mountains" (相同)
```

### 第二步：自动分组

1. 点击顶部导航栏的 **"Prompt 对比"** 标签页（带 ✨ 图标）
2. 点击右上角的 **"自动分组"** 按钮
3. 系统会扫描所有图片，自动创建 Prompt 组

**结果**：
```
✅ 成功创建 X 个 Prompt 组
```

### 第三步：查看对比

1. 在 Prompt 组列表中，找到你想对比的组
2. 点击 **"查看对比"** 按钮
3. 图片会按模型自动分组显示

**对比视图示例**：
```
┌─────────────────────────────────────┐
│ Prompt: a beautiful sunset over... │
├─────────────────────────────────────┤
│ OpenAI - DALL-E 3                   │
│ [图片1] [图片2] [图片3]              │
│                                     │
│ Google - Imagen 3                   │
│ [图片4] [图片5]                      │
│                                     │
│ Midjourney - Midjourney v6          │
│ [图片6] [图片7] [图片8]              │
└─────────────────────────────────────┘
```

## 常见使用场景

### 场景 1：模型选择

**目标**：为特定类型的图片选择最佳模型

**步骤**：
1. 准备 5-10 个代表性的 prompt
2. 在多个模型上生成图片
3. 导入到 sanMediaBox
4. 使用自动分组功能
5. 逐个查看对比，评估效果
6. 记录最佳模型

### 场景 2：Prompt 优化

**目标**：测试不同 prompt 变体的效果

**步骤**：
1. 准备 prompt 的多个变体
2. 在同一模型上生成
3. 导入并分组
4. 对比效果，选择最佳 prompt

### 场景 3：版本对比

**目标**：对比模型不同版本的效果

**步骤**：
1. 使用相同 prompt 在不同版本生成
2. 导入并标记版本信息
3. 使用对比功能查看差异

## 实用技巧

### 技巧 1：规范化 Prompt

为了更好地自动分组，建议：

❌ **不好的做法**：
```
"sunset --ar 16:9 --v 6"  (包含模型参数)
"sunset, midjourney style" (包含模型名称)
```

✅ **好的做法**：
```
"a beautiful sunset over mountains"  (纯描述)
```

将模型参数和风格信息放在 negative_prompt 或 description 中。

### 技巧 2：批量测试

**标准 Prompt 集**：
```
1. 风景类：sunset, mountains, ocean
2. 人物类：portrait, character, face
3. 抽象类：abstract, geometric, pattern
4. 动物类：cat, dog, bird
5. 建筑类：building, architecture, interior
```

使用这些标准 prompt 在不同模型上测试，建立你的模型效果数据库。

### 技巧 3：描述字段的使用

在 Prompt 组的描述中记录：
- 测试日期
- 测试目的
- 结论和发现

**示例**：
```
描述：2026-05-21 测试风景类图片效果
结论：Midjourney v6 在细节表现上最好，
     DALL-E 3 在色彩饱和度上更优
```

## 常见问题

### Q1: 自动分组没有创建任何组？

**A**: 可能的原因：
- 图片没有 prompt 信息
- 没有重复的 prompt（至少需要 2 张相同 prompt 的图片）
- 已经创建过相同的组

**解决方法**：
1. 检查图片的 prompt 字段是否有值
2. 在图片编辑界面手动添加 prompt
3. 确保至少有 2 张图片使用相同 prompt

### Q2: 如何手动创建 Prompt 组？

**A**: 当前版本支持自动分组。手动创建功能将在下个版本提供。

临时方案：
1. 确保图片有相同的 prompt
2. 使用自动分组功能

### Q3: 删除 Prompt 组会删除图片吗？

**A**: 不会。删除 Prompt 组只是删除分组关系，不会删除图片本身。图片仍然保留在收件箱或归档中。

### Q4: 一张图片可以属于多个 Prompt 组吗？

**A**: 可以。系统支持多对多关系，一张图片可以同时属于多个 Prompt 组。

### Q5: 如何导出对比结果？

**A**: 导出功能计划在未来版本中提供。当前可以使用截图工具保存对比视图。
