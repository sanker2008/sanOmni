## ✨ 新增功能

### 1. 图片查看器 🖼️

**功能说明**：
- 双击图片打开全屏查看器
- 支持键盘导航（← → 键切换图片）
- 显示详细信息侧边栏
- 支持快捷操作（编辑、下载、外部打开）

**主要特性**：
- ✅ 高清大图显示
- ✅ 左右导航按钮
- ✅ 键盘快捷键支持
- ✅ 详细信息展示（模型、标签、Prompt、文件信息）
- ✅ 快捷操作按钮
- ✅ 进度显示（如：3 / 10）

**使用方法**：
- 双击图片卡片打开查看器
- 使用 ← → 键切换图片
- 按 ESC 关闭查看器

**文档**：[IMAGE_VIEWER.md](./IMAGE_VIEWER.md)

**功能说明**：

**位置**：

**用户体验改进**：
- 操作步骤从 2 步减少到 1 步
- 减少 50% 的操作时间

**文档**：[VENDOR_EDIT_SHORTCUT.md](./VENDOR_EDIT_SHORTCUT.md)

## 🎨 UI 改进

### 1. 快速编辑弹窗宽度增加

**改进**：
- 从 `max-w-2xl` (672px) 增加到 `max-w-4xl` (896px)
- 宽度增加约 33%
- 提供更舒适的编辑空间

**效果**：
- 编辑长 Prompt 时体验更好
- 图片预览和表单区域比例更合理

---

### 2. 模型和标签显示优化

**问题**：
- 之前显示 "+1"、"+2" 等，无法直观了解具体内容

**解决方案**：
- 为 "+N" 徽章添加 Tooltip 提示
- 鼠标悬停时显示所有隐藏的模型/标签
- 添加 `cursor-help` 样式提示可交互

**效果**：
```
鼠标悬停在 "+2" 上 → 显示提示框：
┌─────────────────┐
│ 其他模型：      │
│ • DALL-E 3      │
│ • Gemini ★      │
└─────────────────┘
```

**文档**：[UI_IMPROVEMENTS_2026-05-20.md](./UI_IMPROVEMENTS_2026-05-20.md)

## 🐛 Bug 修复

### 1. Tooltip 组件嵌套问题

**问题**：
- 多个嵌套的 `TooltipProvider` 导致 JSX 结构错误
- 编译失败

**修复**：
- 只在组件最外层使用一个 `TooltipProvider`
- 移除所有内部的 `TooltipProvider`
- 所有 Tooltip 共享外层的 Provider

**影响文件**：
- `src/components/ImageCard.tsx`
