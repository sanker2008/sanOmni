# 厂商管理快速开始指南

## 🚀 5 分钟上手厂商管理

### 第一步：打开设置

有两种方式：
1. 按键盘快捷键 `Ctrl + ,`
2. 点击应用右上角的设置图标 ⚙️

### 第二步：进入厂商管理

在设置窗口中，点击顶部的"厂商管理"标签页。

### 第三步：查看现有厂商

你会看到系统中已配置的所有厂商，例如：
- Google（2 个模型）
- OpenAI（3 个模型）
- Midjourney（1 个模型）

点击厂商左侧的 `>` 图标可以展开查看模型列表。

---

## 📝 常见操作示例

### 示例 1：添加新厂商 "Stable Diffusion"

1. 点击底部的"添加新厂商"按钮
2. 输入以下信息：
   - **厂商名称**：`Stable Diffusion`
   - **路径标识**：`stable-diffusion`
3. 点击 ✓ 保存图标
4. 完成！新厂商已添加

### 示例 2：为 Stable Diffusion 添加模型

1. 展开刚添加的 "Stable Diffusion" 厂商
2. 点击"添加模型"按钮
3. 输入以下信息：
   - **模型名称**：`SDXL 1.0`
   - **路径标识**：`sdxl-1.0`
4. 点击 ✓ 保存图标
5. 完成！新模型已添加

### 示例 3：修改厂商名称

1. 找到要修改的厂商
2. 点击右侧的 ✏️ 编辑图标
3. 修改名称（例如：`Stable Diffusion` → `Stable Diffusion XL`）
4. 点击 ✓ 保存图标
5. 完成！名称已更新

### 示例 4：删除不需要的模型

1. 展开对应的厂商
2. 找到要删除的模型
3. 点击右侧的 🗑️ 删除图标
4. 在确认对话框中点击"确定"
5. 完成！模型已删除（软删除）

## 💡 实用技巧

### 技巧 1：路径标识的命名规范

路径标识用于文件路径和数据库标识，建议遵循以下规范：

✅ **推荐**：
- `openai`
- `google-imagen`
- `stable-diffusion`
- `midjourney-v6`

❌ **不推荐**：
- `OpenAI`（包含大写）
- `google imagen`（包含空格）
- `stable_diffusion`（使用下划线）
- `图片生成器`（使用中文）

### 技巧 2：批量添加厂商

如果需要添加多个厂商，建议按以下顺序操作：

1. 先添加所有厂商
2. 再逐个展开厂商添加模型
3. 这样可以避免频繁切换

### 技巧 3：查看所有厂商

在"已归档"视图中，左侧厂商分类会显示所有厂商（包括没有图片的），方便你：
- 查看系统中配置了哪些厂商
- 了解每个厂商有多少张图片
- 快速筛选特定厂商的图片

## ⚠️ 注意事项

### 关于删除

- 删除厂商或模型是**软删除**
- 数据仍保留在数据库中，只是标记为不活跃
- 已有图片的关联关系不会被破坏
- 如果误删，可以联系技术支持恢复

### 关于修改

- 修改厂商或模型的名称**不会影响**已有图片
- 已归档的图片会继续显示原有的信息
- 建议在有图片关联前完成配置

### 关于路径标识

- 路径标识一旦创建，建议不要修改
- 如果必须修改，请确保没有图片使用该厂商/模型
- 修改后可能需要重新分类相关图片

## 🆘 遇到问题？

### 问题 1：保存时提示错误

**可能原因**：
- 路径标识包含非法字符
- 路径标识与现有厂商/模型冲突

**解决方法**：
- 使用小写字母和连字符
- 检查是否有重复的路径标识

### 问题 2：删除后厂商仍然显示

**原因**：
- 这是正常的，删除是软删除
- 厂商会从列表中消失，但数据仍在数据库中

**如果需要完全删除**：
- 请联系技术支持

### 问题 3：修改后图片显示的还是旧名称

**原因**：
- 图片记录的是厂商/模型的 ID，不是名称
- 需要重新编辑图片才能更新

**解决方法**：
- 打开图片的快速编辑
- 重新选择厂商和模型
- 保存
