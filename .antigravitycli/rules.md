# Antigravity AI Assistant Rules - Chinese Only

This project requires all AI assistant interactions, code comments, and documentation to be written in Simplified Chinese.

## Rules

1. **User Communication**: Always reply to the user in Simplified Chinese (简体中文).
2. **Documentation**: All planning, implementation, and walkthrough documents must be written in Simplified Chinese.
3. **Code Comments & Docstrings**: Write all inline code comments, block comments, and function/struct/class docstrings in Simplified Chinese.
4. **Code Identifiers**: Standard English should be used for variables, functions, structs, classes, and file names. However, their descriptions, error messages, and logs should be in Chinese where appropriate.
