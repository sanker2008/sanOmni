# Product Overview

sanMediaBox is an AI-powered creative asset management desktop application that combines two independent functional domains under a unified interface: **Prompt Template Management** (sanPromptBox) and **IP Character Management** (sanIPBox).

> **Note**: The two domains are architecturally independent and may be split into separate standalone applications (sanPromptBox and sanIPBox) in the future.

## Core Domains

### Domain 1: Prompt Template Management (sanPromptBox)

Streamline the workflow of collecting, categorizing, and archiving AI-generated images from various platforms (OpenAI, Google, Midjourney, etc.) with intelligent classification and metadata management. Images serve as visual references for prompt templates.

**Key Features:**
- **Smart Classification**: Automatic vendor and model detection from filenames
- **Workflow Management**: Inbox → Tagged → Archived pipeline
- **Metadata Management**: Tags, prompts, negative prompts, and model associations
- **Prompt Groups**: Template grouping and comparison
- **Smart Prompt Rendering**: Intelligent prompt template display with variable support
- **Template Variables**: Editable prompt template variables
- **Vendor & Model Management**: Flexible vendor and model configuration
- **Folder Monitoring**: Auto-import from watched directories

### Domain 2: IP Character Management (sanIPBox)

Manage IP (Intellectual Property) characters and their associated creative assets, including sticker packs, emojis, character sheets, and derivative creations. Supports publishing to various platforms.

**Key Features:**
- **IP Character Profiles**: Create and manage IP character definitions
- **Sticker Packs & Emojis**: Organize character-based sticker and emoji sets
- **Character Sheets**: Manage reference sheets for IP characters
- **Creations Management**: Track derivative works and creations
- **Platform Publishing**: Publish assets to various platforms
- **IP Relationships**: Define and track relationships between IP characters
- **Image Picker**: Select and associate images with IP assets

### Shared Common Features

- **Watermark Handling**: Detection and removal capabilities (including Gemini-powered removal)
- **Image Viewer**: Full-screen browsing with keyboard navigation
- **Settings Management**: User preferences, custom storage paths, theme
- **File System Operations**: File scanning, drag-and-drop, custom inbox/archive paths
- **Trash Management**: Soft-delete with restore capability
- **UI Components**: Shared component library (shadcn/ui based)

## Target Users

- **Prompt Domain**: Users who generate images using AI tools and need a centralized system to organize, tag, and archive their creations with proper metadata tracking.
- **IP Domain**: Creators and designers who manage IP characters and need tools to organize character assets, create sticker packs, and publish to platforms.
