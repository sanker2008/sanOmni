# Product Overview

sanMediaBox is an AI-powered image management desktop application designed to help users organize and manage AI-generated images efficiently.

## Core Purpose

Streamline the workflow of collecting, categorizing, and archiving AI-generated images from various platforms (OpenAI, Google, Midjourney, etc.) with intelligent classification and metadata management.

## Key Features

- **Smart Classification**: Automatic vendor and model detection from filenames
- **Workflow Management**: Inbox → Tagged → Archived pipeline
- **Metadata Management**: Tags, prompts, negative prompts, and model associations
- **Watermark Handling**: Detection and removal capabilities
- **Folder Monitoring**: Auto-import from watched directories
- **Image Viewer**: Full-screen browsing with keyboard navigation
- **Vendor Management**: Flexible vendor and model configuration
- **Custom Storage**: User-defined inbox and archive paths

## Target Users

Users who generate images using AI tools and need a centralized system to organize, tag, and archive their creations with proper metadata tracking.

## Language

Primary language: Chinese (Simplified)
UI and documentation are in Chinese.
